# **ğŸ“Š Overview**

This project presents an interactive Excel dashboard that analyzes the Indian used bike market, focusing on pricing, resale value, mileage, brand mix, ownership patterns, and regional trends.

The dashboard is designed to provide a business-ready, one-page analytical view that helps users understand how factors such as city tier, fuel type, brand, ownership type, and manufacturing year influence bike pricing and resale performance.

Built entirely using Excel Pivot Tables, Pivot Charts, and Slicers, the project demonstrates how Excel can be used as a powerful BI tool without external add-ins.

The dataset was used as-is, focusing on analytical insights rather than heavy data cleaning.
